Technical

Relation to inspring image:

  • Color: The sky, cloud, branch, shadows and the sunset hue color was chosen by me, using a hex code color picker on google.
  • Primitives: The clouds are rendered using 'ellipse', and the shadows of the clouds are render with strokes around the cloud.
  • Polygons: The branche is a single randomly polygon using 'vertex' calls.
  • Composition: Different random seed values will create differenty shaped branches and clouds.
  • Life: The position of clouds is animated over time like if the cloud is drifting on the wind, and it also responds to the horizontal motion of the mouse cursor as well.

Reflection

The project was simple and easy, but also took time and effort due to how we created the design through code and algorithm.
