Description

This project implements evolutionary impressions with 3 different images as a input, both for colored and monochrome images.

Technical

The code uses the inspiring images as a goal for what the evolutionary impression should aim for, and uses some mutation functions in order to render sets of images, each with a score on how close it is to the original image.

Changing the slider for mutation rate allows the rendered images to either be more broader or detailed.

The code can work for both monochrome and colored images, as long as some adjustment is made for each images to allow the code to have mutations more accurate.

This experiment was hard at the start but was simple and easy after the basic codes for mutation was completed.
